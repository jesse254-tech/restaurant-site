# Savanna Kitchen

A responsive restaurant website for Savanna Kitchen — a Kenyan grill &amp; café — built with Bootstrap 5.

## Features

- Responsive Bootstrap navbar with a scroll effect
- Full-screen hero with call-to-action
- About section with quick stats
- Tabbed menu (Starters / Mains / Drinks)
- Photo gallery with hover zoom
- Reservation form
- Footer with opening hours and social links

## Built With

- HTML5
- Bootstrap 5 &amp; Bootstrap Icons
- Custom CSS
- JavaScript

## Running Locally

Open `index.html` in any modern browser, or serve the folder with a local server such as XAMPP and visit `http://localhost/restaurant-site/`.
